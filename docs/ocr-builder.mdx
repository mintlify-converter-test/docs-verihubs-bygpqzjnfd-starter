# Builder

OCR WebSDK instance builder for using OCR WebSDK

## build

Builds the SDK interface that can be started and destroyed.

```declaration declaration
build(): OCRSDK;

```

**Returns:**

| Type | Description |
| --- | --- |
| [`OCRSDK`](ocr-types#ocrsdk) | the SDK interface |

## inspect

Inspects the passed configuration for debugging.

```declaration declaration
inspect(): void;

```

## setAttemptCount

Sets the maximum attempt for the OCR extraction while the SDK is running

```declaration declaration
setAttemptCount(count: number): Builder;

```

**Parameters:**

| Name | Type | Description |
| --- | --- | --- |
| count | `number` | the maximum attempt for the OCR extraction |

**Returns:**

| Type | Description |
| --- | --- |
| [`Builder`](#builder) | this builder |

## setContent

| --- | --- | --- |
| content | [`OCRContentConfiguration`](ocr-types#ocrcontentconfiguration) | the text and image content to replace |

**Returns:**

| Type | Description |
| --- | --- |
| [`Builder`](#builder) | this builder |
