Access [Verihubs Dashboard](https://app.verihubs.com)
, click on DEMO on the sidebar menu. You can easily try to check ID just a click away.

<img className="block" src="images/files_readme_io/_a65d388_image.png" alt="" />

To try DEMO you can follow this steps :

1.  Select which Application Name you want to try this DEMO on. [Make sure you have create one, tho](/docs/applications)
    .
2.  You will be given a quota for each Application Name to try the DEMO..
3.  Select the image of ID card you want to extract.
4.  Click on "**Check**" and the extracted text will be shown on the result.
5.  Click on "**Contact us**" button when you feel like need more information about the service. It will redirect you to Verihubs Representative WhatsApp number.

# ðŸŽˆMANAGE BASIC AUTHENTICATION

MANAGE BASIC AUTHENTICATION is used as an extra authentication when you use callback URL parameter in extract-async API ([https://docs.verihubs.com/reference/extract\_async\_text\_post](/reference/extract_async_text_post)
) .

1.  First, you need to select the application name which will be configured. Next, you can input username and password (min is 8 characters).

<img className="block" src="images/files_readme_io/_bed51d6_image.png" alt="" />

2.  After clicking the create button the encoded authentication token will be appeared at top of the page.

<img className="block" src="images/files_readme_io/_e4855d8_image.png" alt="" />

3.  You can update the existing username and password by selecting the application name and enter the new username and password, then click the update button.
